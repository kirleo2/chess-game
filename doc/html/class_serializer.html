void Serializer::putBit ( bool  bit)

Function to write a bit into a binary file.

This method writes bit on corrent position in current byte array using mask. We can imagine an infinite array of bytes 0x0 (0b00000000), that method fills it up with "1" on correct positions by moving "1" towards the right in mask.
