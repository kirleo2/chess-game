\doxysubsection{Detailed Description}
Class that represents the \doxyref{Game}{p.}{class_game} Engine. 

It has\+: player1, player2 -\/ pointers on 2 players (they can be Local or Computer), gameboard -\/ pointer on chess board, status -\/ current game status, move\+Encoder -\/ instance of class \doxyref{Serializer}{p.}{class_serializer} that is necessary to encode all game moves
