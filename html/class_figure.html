Detailed Description

Abstract class that represents a chess Figure.

It has : icon - fugure's icon isFirstMove - flag if figure has moved chessBoard - pointer to Board. It is necessary for canMove logic colour - colour of the figure is_alive - flag if figure has been killed currentPosition - pair of x and y coordinates
